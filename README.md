# Ibdaa Wheel - عجلة التمكين

مشروع ويب تفاعلي لمدرسة إبداع التكنولوجية – بير المكسور، يتيح للطلاب والمعلمين استخدام عجلة رقمية تفاعلية تحتوي على مجالات تربوية وقيم تعليمية.

يقوم المستخدم بلف العجلة، وعند توقف المؤشر على خانة معيّنة تظهر نافذة تحتوي على شرح تربوي وسؤال للتفكير.

## فكرة المشروع

المشروع مبني على عجلة تربوية تفاعلية مقسمة إلى مجالات رئيسية وخانات فرعية.

داخل العجلة تظهر المجالات الرئيسية، وحول العجلة تظهر الخانات الفرعية. عند دوران العجلة يتوقف المؤشر على خانة فرعية، ثم تظهر نافذة معلومات تحتوي على:

- عنوان الخانة.
- المجال الرئيسي التابع لها.
- شرح مختصر.
- سؤال للتفكير.

## لوحة التحكم

من خلال لوحة التحكم يمكن إدارة المجالات الرئيسية وخانات الدولاب.

### إدارة المجالات الرئيسية

- إضافة مجال جديد.
- تعديل اسم المجال بالعربية.
- تعديل اسم المجال بالعبرية.
- تغيير اللون.
- تغيير الترتيب.
- تفعيل أو إخفاء المجال.
- حذف المجال.

### إدارة خانات الدولاب

- إضافة خانة جديدة.
- اختيار المجال الرئيسي.
- إدخال العنوان بالعربية.
- إدخال العنوان بالعبرية.
- إدخال الشرح بالعربية.
- إدخال الشرح بالعبرية.
- إدخال سؤال للتفكير بالعربية.
- إدخال سؤال للتفكير بالعبرية.
- إضافة أيقونة.
- تغيير الترتيب.
- تفعيل أو إخفاء الخانة.
- حذف الخانة.

## تشغيل المشروع محليًا

### 1. تحميل المشروع

```bash
git clone https://github.com/madadha/ibdaa-wheel.git
cd ibdaa-wheel

2. تثبيت الحزم
composer install


3. إنشاء ملف البيئة
cp .env.example .env

4. إنشاء مفتاح التطبيق

php artisan key:generate

5. إعداد قاعدة البيانات في ملف .env

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ibdaa_wheel_db
DB_USERNAME=root
DB_PASSWORD=

6. تشغيل الجداول

php artisan migrate

7. إدخال البيانات الأولية

php artisan db:seed --class=WheelSeeder

8. تشغيل السيرفر المحلي


php artisan serve
ثم افتح:

http://127.0.0.1:8000/ar

http://127.0.0.1:8000/he

إعداد لوحة التحكم

أضف هذه القيم في ملف .env:

ADMIN_USERNAME=admin
ADMIN_PASSWORD=142536
SESSION_DRIVER=file
CACHE_STORE=file
QUEUE_CONNECTION=sync

ثم افتح:

http://127.0.0.1:8000/admin/login
